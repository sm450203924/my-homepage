# my-homepage

我的个人主页 -*my personal homepage*

##  使用了 -*Build With*
-  前端框架  -*Framework* ： Vue
- 组件库  -*Toolkit*：Element-UI
- CSS扩展语言  -*CSS extension*：SCSS
- 脚本程序设计语言 -*Scripting programming language*: ES6、ES5
- 标准化工具  -*Standard Tooling*：Vue CLI 3
- 可视化类库  -*visualization library*：Echarts
